# 简要总结

## 核心创新点是什么

TM-BSN 根据真实 sRGB 图像中由去马赛克引起的菱形噪声相关结构，提出三角掩码卷积，并通过特征位移和四方向旋转融合，在原始分辨率构造菱形盲区。它还将多个盲区尺度下的教师预测经回注式知识蒸馏转移至轻量 U-Net，实现兼顾精度与速度的自监督去噪。

## 主要实验结论或研究结论是什么

蒸馏后的 TM-BSN 在 SIDD Validation、SIDD Benchmark 和 DND 基准上达到自监督方法中的领先结果；在 DND 的完全自监督设定达到 39.41 dB / 0.949，并较此前最佳 TBSN 提升 0.33 dB。学生模型仅需 3.21 ms 推理，显著快于教师模型，同时更能保留木纹、织物等细粒度纹理。
